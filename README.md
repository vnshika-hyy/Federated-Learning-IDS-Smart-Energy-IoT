🚀 Federated Learning-Based Intrusion Detection for Smart Energy IoT Systems

📌 Project Overview

The integration of IoT devices in smart energy systems has improved automation and efficiency but has also introduced significant cybersecurity risks such as Denial-of-Service (DoS) and False Data Injection attacks.

This project proposes a privacy-preserving Intrusion Detection System (IDS) using Federated Learning (FL), where distributed IoT nodes collaboratively train a global model without sharing raw data.

To enhance robustness against malicious participants, the system integrates adversarial defense mechanisms including robust aggregation and trust-based client evaluation.

🎯 Objectives

Develop a federated learning-based IDS for smart energy IoT environments

Preserve data privacy through decentralized training

Detect cyber attacks using hybrid deep learning models (CNN-LSTM)

Mitigate poisoning and Byzantine attacks using robust aggregation

Evaluate performance under non-IID data distributions
